# Inventory Management System – Backend

This is the backend of an Inventory Management System built using Node.js, TypeScript, Express, MongoDB, and Clean Architecture.

## 🔧 Tech Stack

- Node.js
- TypeScript
- Express
- MongoDB
- Mongoose
- tsyringe (for dependency injection)
- dotenv
- bcrypt
- Clean Architecture Pattern

## 🚀 How to Run Locally

### 1. Clone the repository
```bash
git clone https://github.com/<your-username>/Inventory-manage-server.git
cd Inventory-manage-server/server
npm install
then set up .env variables
PORT=5000
MONGO_URI=your-mongo-uri
JWT_SECRET=your-jwt-secret
start development
npm run dev
